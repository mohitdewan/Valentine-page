# Valentine-page
A cute interactive landing page to ask your partner to be your Valentine ❤️
## Features
- Floating hearts animation
- Background music
- YES button triggers WhatsApp message
- NO button runs away
- Floating photo frames
- Confetti explosion
## How To Use
1. Replace photos in `/images`
2. Replace `music/love.mp3` with your song
3. Update phone number in `script.js`
4. Open `index.html`
## Deploy on GitHub Pages
Settings → Pages → main → root → Save
